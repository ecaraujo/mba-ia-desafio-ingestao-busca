# Desafio MBA Engenharia de Software com IA - Full Cycle

Descreva abaixo como executar a sua solução.

1 - Executar o docker-compose.yml:
    docker compose -f docker-compose.yml up -d

2 - Executar o ingest.py no diretõrio src:
    python ingest.py

3 - Executar o chat.py no diretõrio src:
    python chat.py

3 - Verificar a resposta gerada para a pergunta:
    Liste o nome das empresas de telecom?
